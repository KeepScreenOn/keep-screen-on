## Keep Screen On 是什么？

**Keep Screen On** 是一个基于浏览器的免费在线屏幕常亮工具，也可以理解为
**No Sleep Page**、**Keep Screen Awake Tool** 或 **Screen Always On Website**。
它可以帮助你防止手机、平板、笔记本和电脑屏幕在长时间使用时自动变暗、锁屏或进入睡眠。

当你需要屏幕持续显示内容时，不必修改系统设置，也不需要安装 App、浏览器插件或 APK。
打开网站，开启屏幕常亮模式，让页面保持打开即可。

## 为什么使用 Keep Screen On？

- **立即保持屏幕常亮**：无需下载、无需安装、无需注册账号。
- **浏览器即可使用**：适用于现代手机、平板、笔记本和桌面浏览器。
- **支持多种设备场景**：包括 iPhone、iPad、Android、Samsung Galaxy、
  Windows 笔记本、MacBook、Chromebook 和桌面显示器。
- **适合日常任务**：阅读、做饭、学习、演示、查看仪表盘、下载文件、导航和观看教程。
- **更注重隐私**：屏幕常亮操作在浏览器中完成，不需要提交个人信息。

## 常见使用方式

### No Sleep Page

当你只需要一个简单页面来防止屏幕睡眠时，可以使用
[No Sleep Page](https://keep-screen-on.com/no-sleep-page)。它适合菜谱、笔记、
在线课程、计时器、清单和长文章阅读。

### Keep Screen Awake

当你需要在会议、演示、监控面板、下载任务、构建状态、直播或学习过程中保持屏幕不熄灭时，
可以使用 [Keep Screen Awake](https://keep-screen-on.com/keep-screen-awake)。

### Screen Always On

当你希望屏幕长时间保持显示，例如监控信息、查看操作说明、运行状态页面或做简单的信息展示时，
可以使用 [Screen Always On](https://keep-screen-on.com/screen-always-on)。
